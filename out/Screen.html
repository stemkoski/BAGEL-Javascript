Class: Screen

Screen()

A Screen contains the code for each individual game screen, such as a title screen, menu screens, and screens corresponding to each room or level in the game.
Each screen manages a collection of Groups, each of which stores a list of Sprites, that are rendered by the Game class, in the order in which they were added.
Classes which extend Screen must implement the initialize and update methods.

Constructor

new Screen()

Creates a collection to store Groups, a default Group named "main", a variable to reference the Game containing this screen, and calls the initialize method.
