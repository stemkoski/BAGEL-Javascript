Class: Entity

Entity()

Entity is the super class for Sprite and Group objects.
To be rendered in a Game, entities must be added to the main Group of a Screen, or a secondary Group that has added to the main Group.
Entities store a reference to the Group they have been added to, to simplify removal from the Game (if needed).
