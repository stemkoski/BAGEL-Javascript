Class: Game

Game(width, height, divID)

The Game class sets up the canvas where the game appears, manages a collection of Screen objects, which contain the code for each individual game screen, and manages up the Input object and Clock objects, shared by all screens.
Classes which extend Game must implement the initialize method.
