BAGEL: The Basic Adaptable Game Engine Library

BAGEL is a framework that enables developers to create video games efficiently. It is basic because it is built from first principles and does not make use of any other software libraries. It is adaptable because it can be used to create a wide variety of games.

There is a sample game available here: Starfish Collector
